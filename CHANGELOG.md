# Changelog

## 1.0.0 - 2025-03-10

### Added
- Initial release
- Automatic URL submission to IndexNow-compatible search engines (IndexNow API, Bing, Yandex, Seznam) when entries are saved or deleted
- Queue-based submission with configurable cron job support
- Immediate auto-submit mode for real-time notifications
- API key generation with automatic web root file creation
- Manual submission actions: submit pending queue or all site URLs
- Submission log (last 50 entries, 7-day retention)
- Console commands: `indexnow/submit` (pending) and `indexnow/submit/all` (all pages)
- Full translation support (English and German included)
- Craft Plugin Store licensing
